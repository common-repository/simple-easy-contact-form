== Description ==
"Simple easy contact form" creates contact form in your website.
You can dynamically create contact form inputs and it is configurable from admin side.

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. Go to your plugin manager and activate it.
3. On activation, Simple easy contact form one configuration link is shown in plugin manager and it is also shown in admin menu.
4. Go to Simple easy contact form and manage form inputs that you want to display.
